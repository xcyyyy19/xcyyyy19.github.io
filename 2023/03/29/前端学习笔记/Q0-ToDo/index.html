1.margin，padding，border概念的区分： 这一块可以看一下盒子模型，画个图理解一下

https://developer.mozilla.org/zh-CN/docs/Learn/CSS/Building_blocks/The_box_model

主要涉及的样式其实就是: box-sizing:content-box 以及 box-sizing:border-box

2.outline和border的区别，这一部分实际可以等理解完盒子模型，就能更好理解

主要区别在于:
1.应用场景:一般border能适用于任何元素的样式，outline一般用于表单组件(比如input/textarea)
2.特性不一样：border一般会被计算在元素的宽高大小里面，outline不会
盒模型 - 学习 Web 开发 | MDN
在 CSS 中，所有的元素都被一个个的“盒子（box）”包围着，理解这些“盒子”的基本原理，是我们使用 CSS 实现准确布局、处理元素排列的关键。
